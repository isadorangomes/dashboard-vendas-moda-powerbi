# Dashboard Executivo de Vendas — Varejo de Moda

Projeto de Inteligência de Mercado desenvolvido para um varejista 
de moda com 5 lojas físicas em Curitiba, PR.

## Objetivo
Transformar dados brutos de vendas em informação estratégica 
apresentável, identificando padrões regionais, perfis de 
consumidor e oportunidades de crescimento.

## Período Analisado
Outubro – Dezembro 2023 · 5 filiais · Curitiba, PR

## Resultados Encontrados
- Receita total: R$ 4,4M no trimestre
- Batel lidera com 32,6% da receita e ticket médio de R$ 450
- 26% dos clientes (segmento Premium) geram 63,5% da receita
- Crescimento de +80% de outubro para dezembro

## Segmentação de Clientes
| Segmento | Clientes | Share Receita |
|---|---|---|
| Premium (≥ R$ 2.000) | 684 | 63,5% |
| Recorrente (R$ 800–2.000) | 912 | 27,3% |
| Ocasional (< R$ 800) | 1.029 | 9,2% |

## Tecnologias Utilizadas
- Power BI Desktop (DAX, Power Query, Modelo Estrela)
- Excel (ETL e consolidação de dados)

## Estrutura do Projeto
- `dados/` — base de dados tratada (Excel)
- `imagens/` — prints das 3 páginas do dashboard
- `relatorio/` — arquivo .pbix do Power BI
